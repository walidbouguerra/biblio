# Biblio

A Full Stack Library App using React and Spring Boot.

## Features
- CRUD Application : create, read, update and delete books.

## Front-End
### Framework
- React

### NPM packages
- React Router
- Axios
- Bootstrap
- React Material UI Carousel
- Font Awesome

## Back-End
- REST API : Spring boot + MySQL

### Dependencies
- Spring WEB
- Spring Data JPA
- MySQL Driver
- Lombok
